# Spring
- Framework para crear apps empresariales en la JVM
- Características:
    - Inyección de dependencias
    - Seguridad
    - Acceso a datos
    - Pruebas

# Spring Boot
- Spring pero simplificado para Web Services

| Spring Boot DevTools | Live Reload                                                  |
